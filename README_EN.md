## 🎮 Core Gameplay

Tap the wooden fish = Strike + Merit Accumulation

### Dual Mode Experience

| Classic Mode | Cyber Mode |
|:------------:|:----------:|
| Zen Fish | Neon Merit |
| Serene Atmosphere | Sci-Fi Overload |
| Traditional Aesthetics | Cyberpunk |

### Features

- ⚡ **Real-time CPS** - Clicks per second tracking
- 🔥 **Overdrive Mode** - Triggers "Insanity Mode" when CPS > 5
- 🎯 **9 Wishes** - Wealth, Off-work-on-time, Emotional Stability, Hair Volume, Stock Rise, Looks, Dating, Exam Success, Good Sleep
- 📤 **Share Receipt** - Generates retro thermal receipt-style share images

### Development

```bash
# Clone the project
git clone https://github.com/cstdr/mini-wooden-fish.git
```

### Share Image Logic

Each share generates a random retro thermal receipt-style image containing:
- Current merit count
- Selected wish keyword
- Random "Insanity Literature" copy
- Tilted red "Received Successfully" stamp
