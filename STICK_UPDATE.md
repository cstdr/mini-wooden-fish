## 📐 方向说明

### 旋转原理
```
原始图片:
┌─────────────┐
│  头        │  ← 右上
│            │
│         尾 │  ← 左下
└─────────────┘

旋转180度后:
┌─────────────┐
│        尾  │  ← 左下
│            │
│  头        │  ← 右上
└─────────────┘

等等... 用户要求头在左上角！

让我重新理解...
```

### 重新理解需求

用户说：
- 头在右上角 → 头在左上角
- 尾在左下角 → 尾在右下角

这意味着**顺时针旋转90度**而不是180度！

让我重新检查...

实际上，用户的意思是：
```
原始: 头右上，尾左下
目标: 头左上，尾右下

这是逆时针旋转90度！
```
