## 🐛 故障排除

### 问题 1：还是出现 Timeout

**解决方案**：
```bash
# 1. 完全关闭微信开发者工具
# 2. 删除项目目录下的 .miniprogram 目录（如果存在）
# 3. 重新打开开发者工具
# 4. 重新导入项目
```

### 问题 2：木鱼不显示

**检查**：
1. 控制台是否有其他错误
2. 尝试刷新页面（手指下拉刷新）
3. 检查是否是基础库版本问题

### 问题 3：音效不播放

**可能原因**：
- 音效文件损坏
- 音量设置为 0
- 音频池初始化失败

**解决方案**：
1. 检查控制台是否有音频错误
2. 在设置中关闭再开启音效
3. 如果问题持续，可以暂时关闭音效功能

### 问题 4：震动不工作

**可能原因**：
- 手机系统禁止了震动权限
- 模拟器不支持震动

**解决方案**：
- 在真机上测试震动功能
- 在设置中关闭震动选项

## 🎨 自定义选项

### 更换木鱼图案

如果想用真实图片替换 emoji：

1. 准备一个木鱼图片（PNG/JPG，< 2MB）
2. 放入 `images/` 目录
3. 修改 `pages/index/index.js`：
```javascript
// 替换这一行
woodenFishEmoji: '🐟',
// 改为
woodenFishImage: '/images/your-image.png',
```

### 更换音效

1. 准备一个敲击音效（WAV/MP3，< 500KB）
2. 放入 `audio/` 目录
3. 修改 `pages/index/index.js`：
```javascript
audio.src = '/audio/your-sound.wav';
```
